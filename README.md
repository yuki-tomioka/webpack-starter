# webpack-starter

## 使い方
git clone `https://github.com/yuki-tomioka/webpack-starter.git`
npm ci

## ディレクトリ構成
- dist `開発用ソース出力`
- docs `公開用ソース出力`
- src `入力`

## コマンド
- build:dev `開発用ビルド`
- build:prod `公開用ビルド`
- serve `開発用サーバー立ち上げ`

## 構成モジュール
- webpack-cli
- autoprefixer
- css-loader
- csswring
- html-webpack-plugin
- mini-css-extract-plugin
- postcss-loader
- postcss-nesting
- webpack-dev-server
- webpack-merge

## 開発環境
webpack v4.41.2
node v10.15.3
